# Comsov-Wallet
Comsov Wallet — an anonymous decentralized wallet. Zero-fee P2P transfers, global QR payments, installation without the App Store. Built on 7 core principles of privacy and freedom.

# Sovereign Wallet

**Your money. Your rules. No one sees.**

COMSOV Wallet — полностью анонимный, децентрализованный кошелёк с бесплатными P2P-переводами и глобальной QR-оплатой.

## 7 концепций

- 🔐 **Self-Custody** — ключи только у тебя
- 👁️ **Zero Knowledge** — транзакции не видны никому
- 💸 **No Fee** — P2P-переводы бесплатны
- 📱 **QR Payments** — плати в магазинах по QR
- 🚫 **No Store** — установка без Google/Apple
- 🔄 **DEX** — обмен без посредников
- ⚛️ **Quantum Safe** — защита от будущих угроз
